# Product Showcase App

Questa Ã¨ una semplice webapp frontend sviluppata in React e Javascript che funge da vetrina prodotti, utilizzando la [DummyJSON API](https://dummyjson.com/docs/products) come fonte dati realistica.

## Feature principali

- Lista dei prodotti
- Dettaglio prodotto
- Filtri per categoria e ricerca
- Paginazione sui prodotti
- Stato locale per prodotti preferiti e carrello fittizio (Zustand)
- UI moderna basata su [shadcn/ui](https://ui.shadcn.com) e [Tailwind CSS](https://tailwindcss.com)
- Fetch e caching dati tramite [React Query](https://tanstack.com/query/latest) e Axios
- Router client per navigazione tra homepage, dettagli e carrello (React Router)
- Gestione ottimale di errori, loading e notifiche con toast Shadcn/ui

## Stack tecnologico

- React + JSX
- React Query, Axios
- React Router
- Zustand
- shadcn/ui
- Tailwind CSS
